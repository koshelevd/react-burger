
# React-based "Stellar burgers"

## https://koshelevd.github.io/react-burger/

## Tech
* `React`
* `Redux`
* * `TypeScript`
* `HTML5`
* `CSS modules`
* `DnD`
* `WebSocket`
* `Cypress`

## Install

### Install dependencies
```
$ npm install
```

### Run dev
```
$ npm run start
```

### Prod build
```
$ npm run build
```

### Run cypress tests
```
$ npm run cypress:open
```

### Run int tests
```
$ npm run test
```
