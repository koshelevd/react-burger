
# React-based "Stellar burgers"

## https://koshelevd.github.io/react-burger/

## Tech
* `React`
* `Redux`
* * `TypeScript`
* `HTML5`
* `CSS modules`
* `DnD`
* `WebSocket`
* `Cypress`

## Install
```
$ npm install

$ npm run start

# prod build
$ npm run build

# run cypress tests
$ npm run cypress:open

# run int tests
$ npm run test
```
